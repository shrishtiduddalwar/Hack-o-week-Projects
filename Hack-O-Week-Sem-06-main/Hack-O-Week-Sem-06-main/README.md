Hack-O-Week Progress Report (10 Weeks)

Week 1 – Peak Hour Electricity Spikes
Objective
Identify and predict electricity consumption spikes during peak campus hours to help optimize energy distribution.
Member 1 Tasks
●	Studied energy demand forecasting techniques.
●	Performed time-series analysis on electricity consumption data.
●	Identified patterns such as morning start-up spikes and afternoon peaks.
●	Developed a basic forecasting model using linear regression.
Member 2 Tasks
●	Collected hourly electricity meter readings from multiple campus buildings.
●	Cleaned missing values and removed abnormal readings.
●	Created visualizations showing daily electricity patterns.
●	Built an initial dashboard to monitor peak electricity usage.
Outcome
●	Identified peak demand hours between 9 AM – 1 PM.
●	Created an initial model to predict electricity spikes one day in advance.
